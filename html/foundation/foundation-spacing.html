Spacing

Consistent spacing makes an interface more clear and easy to scan.


Overview

Consistent spacing creates visual balance that makes the user interface (UI) easier for users to scan. Apply consistent spacing to improve the quality of the UI. When several items are in close proximity to each other, they become one visual unit rather than several separate units. Otherwise, their distance should be larger and look more like several visual units.


Size

Our spacing is based on a 10-pixel grid. There are 9 predefined padding sizes. In specific cases, when a huge gap is needed those values can be summed up.


As shown above, we operate on an 10px grid with one half size. For larger space sizes, multiply 40px by any whole number between 2 and 40. (ex. 40px x 2 = 80px, 40px x 3 = 120px)
