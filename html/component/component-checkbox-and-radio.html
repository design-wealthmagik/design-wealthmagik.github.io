Checkbox and Radio

ช่วยให้ user สามารถกรอกข้อมูลเข้าระบบด้วยการเลือก จากตัวเลือกที่กำหนด แบ่งเป็น 2 ส่วนคือ checkboxbox (เลือกได้ทีละหลายตัวเลือก) และ Radio (เลือกได้ทีละ 1 ตัวเลือก)


Types

Types Purpose
Checkboxes ใช้เมื่อมี Action ที่ User สามารถเลือกตัวเลือกได้มากกว่า 1 อัน
Radio ใช้เมื่อมี Action ที่ User สามารถเลือกตัวเลือก ได้เพียงอันเดียว

Spec

Checkboxes

Radio

Usage

1. Checkboxes

ใช้เมื่อมี Action ที่ User สามารถเลือก Choices ได้มากกว่า 1 ข้อ

2. Radio

ใช้เมื่อมี Action ที่ User สามารถเลือก Choices ได้มากกว่า 1 ข้อ

DO

ใช้ Radio สำหรับให้ User เลือกเพียง 1 ข้อเท่านั้น

DON'T

ห้ามใช้ Radio ในการให้ User เลือกตัวเลือกมากกว่า 1 ข้อ
