Tab

Tab จะเอาไว้จัดระเบียบข้อมูลที่มี level ของความสำคัญเท่า ๆ กันหรือใกล้เคียงกัน ให้ User สามารถเห็นได้ทันที


Tab Types

Types Purpose
Normal ใช้สำหรับจัดระเบียบกลุ่มข้อมูลที่เป็นส่วนประกอบในหน้านั้น และจะใช้กับ Background สีขาว
Advanced ใช้สำหรับจัดระเบียบกลุ่มข้อมูลที่มี Level ของข้อมูลใหญ่กว่า Normal Tab และจะใช้กับ Background ที่มีสี เพื่อเน้นให้ Tab นั้นเด่นชัดขึ้น

Usage

1. Normal Tab

ใช้สำหรับจัดระเบียบกลุ่มข้อมูลที่เป็นส่วนประกอบในหน้านั้น และใช้บนพื้นสีขาว เช่นในหน้า Consolidate port

2. Advanced Tab

ใช้สำหรับจัดระเบียบกลุ่มข้อมูลที่มี Level ของข้อมูลใหญ่กว่า Normal Tab และอยู่บนพื้นสี เช่น หน้า Bond Info, Fund Info
