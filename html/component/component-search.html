Search

ช่วยให้ user ทำการค้นหาข้อมูลต่างๆ ที่ต้องการบนเว็บไซต์ ผ่านการพิมพ์ Keyword และให้ระบบทำการค้นหาสิ่งที่ตรงกัน

Wealth Service - Search (Portfolio List)

Search Result/Suggestion Display

จำนวนผล search result ที่แสดงของการ search จะต้องมีความสูงรวมทั้งหมดไม่เกิน 500px และตัวสุดท้ายจะต้องไม่พอดีขอบกล่อง (ควรแสดงให้เห็นตามรูปด้านล่าง ว่าถูกตัดไปบางส่วน) เพื่อให้ user รู้ว่ามีอยู่ข้างล่างอีก ให้ scroll ลงเพื่อดูเพิ่มเติม.

Recent Search

กรณี user กลับมาใช้ search bar ตั้งแต่ครั้งที่ 2 เป็นต้นไป เมื่อ user กดที่กล่อง search ให้แสดง recent search ตามรูป เพื่อช่วยให้ user เข้าถึงการ search ครั้งสุดท้ายได้ง่ายขึ้น
