Input Field

เป็นส่วนที่ให้ user สามารถกรอกข้อมูลที่จำเป็นให้กับระบบโดยการพิมพ์ บางครั้งอาจจะเรียกว่า Textbox หรือ Text field


Stage

Types Purpose
Normal สถานะปกติ พร้อมให้ user ทำการกรอกข้อมูล
Active สถานะกำลังใช้งาน เมื่อ user กำลังกรอกข้อมูล
Error สถานะ Error เกิดขึ้นเมื่อ user กรอกข้อมูลไม่ตรงตามที่กำหนด
Disable สถานะไม่พร้อมใช้งาน และ user ไม่สามารถใช้งานหรือกรอกข้อมูลได้

Usage

1. Labels คือสิ่งสำคัญ

Labels ทำหน้าที่เป็นคำสั่งที่บอกให้ User รู้ว่าต้องกรอกอะไรลงใน Input Field นั้น

2. ใช้ Placeholder Text เพื่อขยายความให้ชัด

ใช้ในกรณีที่ไม่มี Label หรือต้องการขยายรายละเอียดเกี่ยวกับ Text fields นั้น เช่น กล่อง Search

3. ใช้ Field จัดชิดซ้าย (Left-Align) สำหรับข้อมูลทั่วไป

ใช้สำหรับ text field ทั่วไป เช่น หน้าสร้างแผนการลงทุน

4. ใช้ Field จัดชิดขวา (Right-Align) เมื่อต้องการเปรียบเทียบ

ใช้กับการกรอกข้อมูลพวกตัวเลข ที่ต้องมีการเปรียบเทียบค่าในช่องที่กรอกข้อมูล เช่น หน้าตั้งสัดส่วนการลงทุน
